Thank you for downloading Rocky the Rocket!

After downloading, make sure all files are extracted in the same directory. Then, double-click on the Rocky.exe Application File to run the game.

The controls are as follows:

Space Bar - Launch
Up Key - Move Forward
Down Key - Move Backward
Left Key - Move Left
Right Key - Move Right
Esc - Quit Game

There will be more updates in the future to make the game more enhanced. Please read the release notes to find out more. Thank you and fly high :)
